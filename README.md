# Java Calculator

A simple calculator application built using Java.  
This project performs basic arithmetic operations like:

- Addition
- Subtraction
- Multiplication
- Division

## Features

- User-friendly console interface
- Performs basic mathematical calculations
- Handles invalid input
- Lightweight and beginner-friendly project

## Technologies Used

- Java
- VS Code / IntelliJ IDEA / Eclipse

## Note

Some parts of the project UI/design were built with the help of AI tools for faster development and learning purposes.  
However, the complete code, logic, and working of the project were fully understood, reviewed, and implemented carefully during development.
